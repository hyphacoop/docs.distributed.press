# Supported Protocols
See this helpful blog post by Mauve on [comparing peer to peer protocols](https://blog.mauve.moe/posts/protocol-comparisons)

### Hyper/Holepunch
[Documentation](https://docs.holepunch.to/)

> A distributed, secure append-only log is a useful tool for creating fast and scalable applications without a backend, as it is entirely peer-to-peer.

### IPFS
[Documentation](https://docs.ipfs.tech/)

> The InterPlanetary File System (IPFS) is a set of composable, peer-to-peer protocols for addressing, routing, and transferring content-addressed data in a decentralized file system. Many popular Web3 projects are built on IPFS.
