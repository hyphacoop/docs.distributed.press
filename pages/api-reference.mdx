# API Reference
How to use the Distributed Press API to publish your website content and the Distributed Press API for your project

<APIWarning />

There are 3 main types of resources that the API manages:
- [Sites](./api-reference/site): Manages a site's static files and configuration (which protocols to seed to, domain, etc.)
- [Publisher](./api-reference/publisher): Users which can manage individual site deployments
- [Admins](./api-reference/admin): Users which can create, modify, and delete publishers. They also can do anything a publisher can
